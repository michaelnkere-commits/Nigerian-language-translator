


# COS 101 African Language Translator
![Screenshot_15-1-2026_05928_github com](https://github.com/user-attachments/assets/32b53263-8bff-4208-9dbb-cb51a7e38633)


## Description
This is a simple Python program developed as a COS 101 group project.  
The program translates English words into selected African/Nigerian languages using Python dictionaries.

## Supported Languages
- Igbo  
- Yoruba  
- Hausa  
- Edo  
- Idoma  

Each language contains 20 English words and their translations.

## How It Works
1. The user selects a language from the menu.
2. The user enters an English word.
3. The program displays the translation if the word exists.

## How to Run
Ensure Python 3 is installed, then run:

```bash
python translator.py
